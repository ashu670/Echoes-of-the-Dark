# Echoes of the Dark

A first-person psychological horror prototype built in Unity HDRP.

---

## Overview

Echoes of the Dark is focused on creating tension through **player-driven mechanics**.
Movement generates noise, and that noise directly affects how the enemy reacts.

The project is currently in a prototype stage, validating core gameplay systems.

---

## Features

* First-person movement system (walk, run, jump)
* Noise-based gameplay (`noiseVal` affects enemy behavior)
* Enemy AI with:

  * Sound detection
  * Vision (line-of-sight)
  * Delayed reaction
  * Last known position tracking
* Basic kill system on proximity
* HDRP environment with fog-based visibility

---

## Tech Stack

* Unity (HDRP)
* C#
* NavMesh (AI movement)

---

## Project Status

Prototype in development.
Core gameplay loop is implemented and being expanded.
